What it Does

Incentivize behavior with a currency with actual value.

How it Works

Using our coin designer, we will walk you thourgh the creation of your new currency. You'll be able to:

  • Give it a name
  • Cap the total supply
  • Define if/how new tokens are generated
  • And more...

We provide an API library in the most popular programming languages for you to implement the currency in your game.


#!/bin/python

import Payvr

wallets = {
    Bob: '841fd7d47d50406fa1c31558bb',
    Alice: '56592fb3c396713d10954d62c8465fc4f'
}

bank = Payvr('my-super secret-API-key')
bank.give(100, wallets['Bob'])

bob = bank.getUser('841fd7d47d50406fa1c31558bb')
print(bob.balance())
> 100

bob.transfer(60, wallets['Alice'])
print(bob.balance())
> 40
